Este es un bot que estoy utilizando mientras busco mi primera experiencia laboral. Lo creé para automatizar el proceso de revisar todas las páginas que generalmente reviso buscando ofertas. Además, hicé que me enviara las ofertas por correo electrónico, por lo que puedo leerlas en cualquier dispositivo y en cualquier momento.

Esta versión del bot fue modificada extrayendo información personal.
Las variables a rellenar con dicha información están definidas con valores explicativos de cada campo.
